Creating a quote

Quotes are an essential part of winning work and setting clear expectations with your customers. In hipages tradiecore, you can easily create detailed, professional quotes that include labour, materials, and optional deposit requirements and send them directly to your customer via email.

You can also add line items, apply tax settings, and include attachments if needed. Once the customer accepts the quote or pays the deposit, the job status will automatically update to Accepted.

Creating a quote

Option 1: Creating a quote from a job (recommended)

  1. Select Jobs from the navigation bar.

  2. Choose the job you want to quote for.

  3. Select add in the Quotes section.

Using this method pre-fills the customer’s name and job details based on the information already saved.

Option 2: Creating a quote from the Quotes list

  1. Select Jobs → Quotes.

  2. On the Quotes list screen select, Add to open a blank quote screen.

  3. Enter the customer’s details, quote description, and tax method manually.

Viewing and Changing Quote Details

To update or change customer details:

  1. Select the customer name at the top of the quote.

  2. Select Edit.

  3. From here, you can either:

    • Update the existing customer’s name, phone number, email, or address

    • Select Search Existing Customers to find and select a different customer from your list

To update the quote description:

  1. Select the Quote Description row.

  2. Edit the text to describe the job in your own words — clearly and concisely.

To update tax preferences:

  1. Select the Tax field in the quote.

    A pop-up will appear with tax options:

    • Tax Inclusive (default): Amounts entered already include GST. The GST will be calculated and shown separately but not added again.

    • Tax Exclusive: GST is calculated and added on top of the amounts you enter. [See: Tax Inclusive and Exclusive]

  2. Choose the setting that best matches how you usually quote.

Adding items to your quote

  1. In the Items section, select add item .

  2. In the Name field, type the name or code of the item.

    • If you’ve used the item before, select it from the list of saved items.

    • To create a new item, select Create a new line item. [See: Creating a new item for more details.]

  3. Adjust the following if needed:

    • Quantity: Number of units

    • Unit Price: Price per item

    • Tax Rate: Default is 10% GST. Select None if no GST applies.

    The Total and GST amounts will be calculated automatically.

  4. Select Add to include the item in the quote.

    If you select Cancel, the item won’t be saved.

Each item you add appears in the Items section. The Subtotal and GST for the full quote will update automatically.

Note:

Changes made to an item here will override the saved version of that item once the quote is saved. If you need different versions of the same item, create new items with unique names.

Adding a deposit (optional)

If you want to collect a deposit before work begins:

  1. Go to the Deposits section.

  2. Enter the deposit amount. [See Adding deposits to a quote for more information]

[See: Adding deposits to a quote video]

Final steps

  • To open the related job, select it from the Associated Job section.

  • Select Send to email the quote to your customer [see: Email an invoice or quote to a customer], or

  • Select Save to keep it as a draft.
